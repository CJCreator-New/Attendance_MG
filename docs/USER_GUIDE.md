# User Guide

## Getting Started

### 1. Login
- Open the application
- Enter your email and password
- Click "Login"

### 2. Dashboard
After login, you'll see the dashboard with:
- Total employees count
- Present/Absent statistics
- Recent activity
- Quick actions

---

## Employee Management

### Add Employee
1. Click "Employees" in sidebar
2. Click "Add Employee" button
3. Fill in details:
   - Employee ID (unique)
   - Name
   - Gross Salary
   - Opening CL Balance
   - Department
4. Click "Save"

### Edit Employee
1. Find employee in list
2. Click "Edit" button
3. Update details
4. Click "Save"

### Delete Employee
1. Find employee in list
2. Click "Delete" button
3. Confirm deletion

---

## Attendance Management

### Create Attendance Sheet
1. Click "Attendance" in sidebar
2. Select month and year
3. Click "Create Attendance Sheet"
4. Sheet is generated with all employees

### Mark Attendance
1. Click on any cell in the attendance table
2. Select attendance code:
   - **P** - Present
   - **A** - Absent
   - **CL** - Casual Leave
   - **WO** - Week Off
   - **PH** - Public Holiday
   - **HP** - Half Present
   - **OD** - On Duty
   - **WFH** - Work From Home
3. Attendance is saved automatically

### Bulk Update
1. Select multiple cells (coming soon)
2. Choose attendance code
3. Apply to all selected

## Salary Management

### View Salary
1. Click "Salary" in sidebar
2. Select month
3. View salary breakdown for all employees

### Salary Components
- **Basic:** 50% of earned gross
- **DA:** 25% of earned gross
- **HRA:** 25% of earned gross
- **EPF:** 12% of (Basic+DA) if gross > ₹21,000
- **ESI:** 0.75% of earned gross if gross ≤ ₹21,000
- **PT:** ₹200 if earned gross > ₹21,000

### Add Bonus/Deductions
1. Click on employee row
2. Enter bonus amount
3. Enter other allowances
4. Enter other deductions
5. Click "Save"

### Export Salary Slip
1. Click "Export" button
2. Select format (Excel/PDF)
3. Download file

---

## Leave Management

### Apply Leave
1. Click "Leave" in sidebar
2. Click "Apply Leave"
3. Fill in details:
   - Leave type (CL/SL/EL)
   - Start date
   - End date
   - Reason
4. Click "Submit"

### Approve/Reject Leave
1. Go to "Pending Leaves"
2. Click on leave request
3. Review details
4. Click "Approve" or "Reject"
5. Add comments (optional)

### View Leave Balance
1. Click on your profile
2. View "Leave Balance" section
3. See available leaves by type

---

## Reports

### Generate Report
1. Click "Reports" in sidebar
2. Select report type:
   - Attendance Summary
   - Salary Summary
   - Leave Summary
3. Select date range
4. Click "Generate"

### Export Report
1. After generating report
2. Click "Export" button
3. Choose format (Excel/CSV/PDF)
4. Download file

## Settings

### Profile Settings
1. Click profile icon
2. Click "Settings"
3. Update:
   - Name
   - Email
   - Password
   - Photo

### Notification Settings
1. Go to Settings > Notifications
2. Enable/disable:
   - Email notifications
   - SMS notifications
   - Push notifications
3. Choose notification types:
   - Attendance reminders
   - Leave approvals
   - Salary updates

### System Settings (Admin only)
1. Go to Settings > System
2. Configure:
   - Company details
   - Working days
   - Public holidays
   - Leave policies

## Troubleshooting

### Can't Login
- Check email/password
- Clear browser cache
- Contact admin

### Attendance Not Saving
- Check internet connection
- Refresh page
- Try again

### Salary Calculation Wrong
- Verify attendance data
- Check salary configuration
- Contact HR
